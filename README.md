# "Background Audio Keepalive" Extension Guide

## Installation on Google Chrome
1. Go to `chrome://extensions`.
2. Enable "Developer mode".
3. Click "Load unpacked" and select the folder: `c:\Users\oliwi\Videos\Qunlock`.

## Features
- **Audio Keepalive**: Prevents TikTok, Instagram, and Facebook from muting audio in the background.
- **Popup**: Click the extension icon to view status.
- **Language Selection**: You can change the language in the popup menu to one of many available (English, Polish, German, French, Italian, Spanish). Your choice is saved.

## Update
If the extension is already installed, click the refresh icon on the extensions page to load new changes (icon, popup, languages).
